# streamConnectViz
Interactive Web-Based Project to Visualize Stream Connectivity Image Data

How to RUn:

1. Run ```npm install``` to install packafges needed to run the program (such as react-leaflet)

2. Run ```npm run start``` from the command line to open the program in a abrowser window

## Instructions on How to Use the Program

1. Select the Calender on the tab bar on the right to add dates
<img width="1663" alt="Screenshot 2023-05-10 at 10 07 10 PM" src="https://github.com/jaredrkozar/streamConnectViz/assets/59402943/278ceb0a-d247-48ef-afda-0a5402ab0250">

2. Add a date by clicking the calender icon with a plus on it

<img width="486" alt="Screenshot 2023-05-10 at 10 12 09 PM" src="https://github.com/jaredrkozar/streamConnectViz/assets/59402943/80ca40fb-9dbf-4e40-b840-65748fd680aa">

3. Select a date you want to view the locations for by clicking on the Year picker to select a year and the Date picker to select a specfici date from that year

<img width="690" alt="Screenshot 2023-05-10 at 10 14 50 PM" src="https://github.com/jaredrkozar/streamConnectViz/assets/59402943/8bab7bdd-f012-4c60-8ebe-7f15fd7b330b">

4. Click the Add Date button to add the date to the list. The map should refresh automatically, and if you have multiple dates selected, the map shows all of the locations that are available for those dates. To remove a date from the list of selected dates, click on the x button to the right of the date. The map will refresh automatically with all of the new locations that are avilable for these new dates.

<img width="1203" alt="Screenshot 2023-05-10 at 10 16 53 PM" src="https://github.com/jaredrkozar/streamConnectViz/assets/59402943/ad7c7c88-efb0-4fa4-98db-d82d5a17e82c">

5. Select a location by clicking on the icon. A popup will appear that lets you select which date you want to view the images for. You can select any of that dates that you have picked in your dates list (see step 4)

<img width="987" alt="Screenshot 2023-05-10 at 10 12 24 PM" src="https://github.com/jaredrkozar/streamConnectViz/assets/59402943/2145e948-4ba1-4346-b951-3b00a02f510b">

6. Select the submit button to add the location with the select date to the list of selected locations. To remove a location from the list of selected locations, click on the x button to the right of the location.

<img width="1191" alt="Screenshot 2023-05-10 at 10 19 48 PM" src="https://github.com/jaredrkozar/streamConnectViz/assets/59402943/8240d8e3-94af-4ab5-9de3-7f0b4ca751a4">

7. Select the Get Images button to gett all images for the locations you selected at the dates you selected
